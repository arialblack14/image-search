Image Search Abstraction Layer

Just another FreeCodeCamp API Project

A project for the Freecodecamp curriculum.

Objective

Build a full stack JavaScript app that does the following:

  • search for images like this
  • browse recent search queries like this

Usage #1

In order to get the image URLs, alt text and page urls relating to a given search string, user must navigate to '/api/images/' and then type his search.

Usage #2

User can paginate through the responses by adding a ?offset=2 parameter to the URL.

User Story

I can get a list of the most recently submitted search strings.
