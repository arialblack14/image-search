Image Search Abstraction Layer

Just another FreeCodeCamp API Project

A project for the Freecodecamp curriculum.

Objective

Build a full stack JavaScript app that does the following:

  • search for images like this
  • browse recent search queries like this

Usage #1

In order to get the image URLs, alt text and page urls relating to a given search string, user must navigate to '/api/images/' and then type his search. Use can also add '?query=x' where x is the offset, in order to get to a certain page of results.

Usage #2

User can paginate through the responses by adding e.g. ?offset=2 parameter to the URL.

User Story

I can get the 10 most recently submitted search strings by navigating to '/api/latest/imagesearch'.
